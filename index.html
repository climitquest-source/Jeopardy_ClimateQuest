How to host

  1. Enter team names (up to 4). Each team has a keyboard buzzer: A, L, W, P. On-screen buttons work too.
  2. Click a dollar tile to reveal a clue. Open buzzers when you finish reading.
  3. First buzz locks others. Mark Correct/Wrong to score. Pass lets others buzz.
  4. Daily Double: only the selecting team answers after a wager.
  5. Final Question (optional): enter wagers, reveal, then mark correct/wrong per team.

Tip: connect a laptop to a projector. This is a static app; no accounts, no backend.
